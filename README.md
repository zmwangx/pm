*Man page editing made less painful.*

## Prerequisites

- A C++11 compiler;
- A POSIX system with `man(1)`;
- Python 3.3 or later;
- A modern web browser with support for server-sent events
  ([no Microsoft for you!](http://caniuse.com/#feat=eventsource)).
